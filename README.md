# expresso

Começe a desenvolver os seus projetos em Data Science com o Expresso.

Ambiente criado para que você possa mergulhar no mundo do data science.

O ambiente usa o docker para criar o seu ambiente.

## Requisitos

- MacOS ou Linux
- Docker
